# Devenez le CTO de Votre Santé Posturale

## Description du projet
Application web 100% frontend permettant aux utilisateurs de pratiquer des mouvements sportifs de base (squats, pompes, yoga, etc.) en respectant une posture correcte pour prévenir les blessures.  
L’application fournit des instructions personnalisées basées sur un profil sportif défini via un **QCM côté client**, et illustre les mouvements avec des images ou animations.  

## Fonctionnalités principales
- Création du profil sportif via QCM (niveau, sports pratiqués, objectifs).  
- Instructions personnalisées pour chaque mouvement sportif.  
- Illustrations ou animations pour montrer la posture correcte.  
- Sélection de mouvements : squats, pompes, yoga, etc.  
- Recommandations de produits Decathlon (optionnel).  
- Interface moderne et responsive réalisée avec **React + Vite**.  

## Prérequis
- Node.js >= 18.x  
- npm ou yarn
